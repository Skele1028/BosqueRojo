Juego de Mesa

Instrucciones de Juego

Para moverse el jugador debe lanzar un dado correr tantas casillas como el dado le indique a menos que salga 5 o 6, en ese caso:

5: Aparece un enemigo

6: El jugador cae en una trampa

Combate:

En el momento que el jugador entra en combate, se usa un sistema de turnos iniciando por el jugador, el cual debe escoger qué habilidad usar, y lanzar el dado de acuerdo a lo que indique la tarjeta, luego el enemigo atacará al jugador infligiendo daño.

El jugador se puede encontrar con las siguientes casillas:

Casilla de acción: Si el jugador cae en una casilla de acción debe tirar de nuevo el dado y según el número que obtenga el jugador obtiene estrellas (★) por realizar daño a elementos humanos.

Casilla de recuerdo: El jugador realiza el lanzamiento de un dado y obtiene puntos por estar en esta casilla obligatoria, depende del número en el dado se obtienen estrellas (★) o la posibilidad de tener un recuerdo ®. 1: 1★ 2 - 3: 1★, 1® 4 - 5: 2★, 1® Cuando el jugador obtiene el último recuerdo antes de enfrentarse al primer jefe final, el jugador obtiene la tarjeta con la habilidad “Fuego” Cuando el jugador obtiene el segundo recuerdo después de enfrentarse al primer jefe final, el jugador obtiene la tarjeta con la habilidad “Agua”

Casilla Roja: Aquí el jugador se encuentra con el jefe final del nivel.

Si el jugador cae en una trampa deberá rodar la pirinola y enfrentarse a 3 tipos de trampa:

1. Una trampa colgante cae de una rama de un árbol cercano

2. Se activa un detector de elementales

3.Aparecen 3 humanos que generan daño inmediato

Habilidades:

Para usar las habilidades el jugador debe tirar nuevamente el dado para poder:

Huir: puede ser usada para escapar de los enemigos a menos que el jugador saque el valor 1.

Fuego: ATAQUE: Daño enemigo -> 2 x dado / Daño propio -> 1 x dado DEFENSA: Curación -> 1 x dado / Reducción de Daño -> 3

Agua: ATAQUE: Daño enemigo -> 1 x dado / Daño propio -> -2 DEFENSA: Curación -> 2 x dado / Reducción de Daño -> 2

Estrellas:

La puntuación será con estrellas (★), cada estrella equivale a 100 puntos. 2 ★ : Eliminar un enemigo 5 ★ : Derrotar a un jefe final x ★ : Según lanzamiento en casillas de acción.
